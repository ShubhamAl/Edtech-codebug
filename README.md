# Campus++

Campus++ is a faculty/admin-focused academic platform built with Next.js.

> Note: Student authentication and student dashboard flows have been removed. This application now supports institute, faculty, and admin access only.

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Main Access Routes

- Institute login: `/institute-login`
- Institute register: `/institute-register`
- Faculty area: `/faculty`

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Deployment

Deploy as a standard Next.js app on platforms like Vercel.

For deployment details, see [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying).
